# Guess Number Game
This is a simple number guessing game written in
Python.
## Description
The computer chooses a random number between 1 and 20.
The player tries to guess the number.
## How to run
Run the following command:
`python game.py`
## Features
- Random number generation
- User input
- Simple if/else condition
- Win or lose message
